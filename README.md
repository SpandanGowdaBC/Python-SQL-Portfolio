# 🐍 Python + SQL Full Stack Portfolio

This repository contains three backend automation projects demonstrating **OOP Principles**, **SQL Database Management**, and **Software Testing** methodologies.

## 📂 Project 1: Parking Allocation System With VIP System(SQL + OOP)
**Role:** Developer & Tester
- **Logic:** Automates vehicle entry/exit using Class Inheritance.
- **Database:** SQLite database tracks timestamps and VIP status.
- **Key Feature:** Implemented a Polymorphic function for 'VIP Surcharge' calculation (+50% rate).
- **Testing:** Validated boundary cases for "Full Parking" and "Duplicate Entry" handling.

## 📂 Project 2: Employee Leave Management (CRUD)
**Role:** Backend Logic Designer
- **Logic:** Manages leave balances with strict business rules.
- **Database:** Handles efficient batch updates for end-of-month processing.
- **Key Feature:** Automated "Rollover Logic" using SQL `MIN()` functions to enforce a 30-day cap.

## 📂 Project 3: E-Commerce BOGO Engine (Algorithms)
**Role:** Algorithm Designer
- **Logic:** Dynamic cart calculation fetching data from a Product Catalog.
- **Database:** Pre-seeded Product Table to simulate real inventory.
- **Key Feature:** "Buy-One-Get-One" sorting algorithm ensuring the cheapest item in a pair is free.

---
### 🛠️ Tech Stack
- **Language:** Python 3.x
- **Database:** SQLite3
- **Concepts:** OOP (Inheritance, Polymorphism), Exception Handling, File I/O
